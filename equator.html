EQUATOR Benchmark

A deterministic reasoning benchmark built as a true blind study with scientific rigor: fixed protocols, preregistered analysis, and binary, auditable criteria.

Design
  • Blind evaluation: annotators unaware of model identity
  • Binary criteria: correctness, completeness, groundedness
  • Predefined prompts and seeds; no cherry‑picking
Statistics
  • Stratified sampling by domain and difficulty
  • Wilson intervals for binary outcomes
  • Multiple‑comparison control (Holm)
Reproducibility
  • Dataset/version locking with manifest
  • Per‑trial logs and hashable configs
  • Open protocol; third‑party replication encouraged

Protocol (v1)

  1. Preregister evaluation plan: tasks, sample size, metrics.
  2. Freeze dataset release (manifest.json) and split.
  3. Generate answers per model with fixed seeds and retries.
  4. Blind adjudication by small local LLMs + human tie‑breakers.
  5. Score binary criteria; log rationales and evidence.
  6. Aggregate by stratum; compute CIs and significance.
  7. Publish report and raw logs with redactions.

Scope

  • Domains: math word problems, tool use, retrieval grounding, safety.
  • Task types: open‑ended QA, stepwise reasoning, code explanations.
  • Non‑goals: synthetic multiple‑choice without reasoning traces.
