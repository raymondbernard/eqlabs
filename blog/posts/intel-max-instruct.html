Instruct on Intel Max Series GPUs

Notes on training and inference on Intel accelerators.

Overview

A lightweight instruct pipeline targeting Intel Max Series GPUs with focus on portability and performance. Covers environment setup, model compatibility, and reproducible evaluation with EQUATOR.

Setup

  • Install drivers and oneAPI runtime/toolkits
  • Prepare Python env and dependencies
  • Verify device visibility and memory

Training & Inference

Demonstrates instruct fine‑tuning and optimized inference paths. Measure throughput, latency, and accuracy with deterministic criteria.

Reproducibility

Use fixed seeds, pinned versions, and dataset versioning. Store EQUATOR logs per run to compare hardware/software settings over time.
